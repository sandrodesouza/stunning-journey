# Stunning Journey React

### development

copy .env.demo into .env file on the root folder.

```bash
yarn install
yarn start
```

### testing

copy .env.demo into .env file on the root folder.

```bash
# unit tests
yarn test
```

### build for prod

create .env file on the root folder with constrains needed. Check .env.demo for example.

```bash
yarn install
yarn build
```
